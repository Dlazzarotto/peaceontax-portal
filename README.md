# PeaceOnTax Portal
Secure client portal for Peace on Tax — peaceontax.com

**Stack:** Next.js 14 · TypeScript · Supabase · Claude AI · Tailwind · Vercel

## Setup
1. Copy `.env.example` to `.env.local` and fill in your keys
2. Run SQL files in Supabase SQL Editor
3. Deploy to Vercel with Node.js 20 and `npm install --legacy-peer-deps`
